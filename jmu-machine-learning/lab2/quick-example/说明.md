# 决策树实验

快速实现决策树算法实验，包含数据准备、模型对比、参数调优和最优模型评估。

## 环境要求

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

## 快速开始

### 方式1：一键运行（推荐）

```bash
cd quick-example
python run_all.py
```

### 方式2：逐步运行

```bash
cd quick-example
python 00_prepare_data.py
python 01_compare_criteria.py
python 02_parameter_tuning.py
python 03_final_model.py
```

## 实验内容

1. **数据准备** - 加载Wine数据集，探索性分析
2. **划分标准对比** - 对比Gini vs Entropy
3. **参数调优** - 系统测试max_depth、min_samples_split等参数
4. **最优模型** - 网格搜索最优参数，评估最终模型

## 输出文件

- `README.md` - 完整实验报告（自动生成）
- `output/figures/` - 所有实验图片

- `output/best_model.npz` - 最优模型参数

# Wine 数据集实验规划

## 实验目标
使用 Wine 葡萄酒质量数据集，完成 sklearn 机器学习分类实验

## 数据集
- **Wine Dataset** (来自 sklearn 内置数据集)
- 特征：葡萄酒的理化指标（酒精度、酸度、镁含量等 13 个特征）
- 标签：葡萄酒类别（class 0/1/2，共 3 类）

---

## 脚本规划

### 01_load_data.py - 数据加载与预处理
**功能**：
- 加载 sklearn 内置 wine 数据集
- 数据标准化（StandardScaler）
- 保存处理后的数据

**出图**：
- 📊 特征分布直方图（13个特征的数值分布）
- 🥧 标签类别饼图（class 0/1/2 占比）
- 🔥 特征相关性热力图

---

### 02_split_data.py - 划分数据集
**功能**：
- 普通划分（多次）vs K折交叉验证
- 展示两种方法的区别

**出图**：
- 📈 普通划分结果波动图（10次划分的精度散点/折线图）
- 对比展示 K折的稳定性

---

### 03_train_models.py - 多模型训练（5个模型）
**功能**：5个分类模型，按推荐程度排序：

| 推荐度 | 模型 | 说明 |
|:---:|------|------|
| ⭐⭐⭐⭐⭐ | **Random Forest** | 集成方法，稳定且效果好 |
| ⭐⭐⭐⭐ | **SVM (RBF核)** | 对中小数据集表现优秀 |
| ⭐⭐⭐⭐ | **Gradient Boosting** | 梯度提升，通常效果最好 |
| ⭐⭐⭐ | **Logistic Regression** | 线性基线模型，易解释 |
| ⭐⭐⭐ | **KNN** | 简单直观，适合入门 |

**出图**：
- 📊 各模型准确率对比柱状图

---

### 04_evaluate.py - 模型评估
**功能**：
- K折交叉验证详细评估
- 混淆矩阵分析

**出图**：
- 📈 K折各折精度折线图
- 🔥 各模型混淆矩阵热力图

---

### 05_bias_variance.py - 偏差-方差分析（附加题）
**功能**：
- 计算各模型的 bias 和 variance
- 分析过拟合/欠拟合

**出图**：
- 📊 偏差-方差对比柱状图
- 🎯 Bias-Variance Tradeoff 曲线（可选）

---

## 文件结构
```
wine/

├── 02_split_data.py      # 划分数据集 + 可视化
├── 03_train_models.py    # 多模型训练 + 对比图

├── 05_bias_variance.py   # 偏差-方差分析
```

按顺序运行 01 → 02 → 03 → 04 → 05，最后整理实验报告
